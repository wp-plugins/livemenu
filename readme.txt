**Build Your Menus Live**

Livemenu is an easy to use plugin for building beautiful, customized menus for your site. With no configuration needed, tons of options to choose from, and the ability to create & style your menu LIVE.
Add, Edit, Remove & Re-order Menu Items on your front-end !

**Live Style**

You will love the ease of use of the style system provided with this plugin. You could customize almost everything live with livemenu.

**Mega Content**

With the responsive grid system built in, livemenu can add widgets, shortcodes, mega links & custom HTML on the same submenu.
Set full width or custom width for each mega submenu & you can configure the width of each column.

== Installation ==
Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
